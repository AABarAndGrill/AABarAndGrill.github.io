TWRP

info
Note: This is a data/media device! Please read: What is EXCLUDED from a TWRP backup?
  • Common TWRP (for GT-P3100, GT-P3110, GT-P3113, GT-P5100, GT-P5110, GT-P5113)

    • The common TWRP is compatible with GT-P3100, GT-P3110, GT-P3113, GT-P5100, GT-P5110 and GT-P5113 at the same time.

    info
    Note: The new "Common TWRP" won't be able to flash Roms compiled before 22.03.2016. For Roms compiled since 24.09.2016 it's recommend to use TWRP 3.0.2-6 or newer.

Device specific TWRP for Galaxy Tab2 (discontinued, but they should still work to flash latest Roms): GT-P3100, GT-P3110 / GT-P3113, GT-P5100, GT-P5110 / GT-P5113
