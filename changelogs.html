--- title: Changelogs for Devs ---

When I started working on android I had a lot of luck: I learned from great people who love OpenSource. I don't like to hide changes and stuff we have been working on.


Here's some changelogs which might help other devs getting an OMAP4 device running on any other custom rom:

Android 4.4.4 R2.0.1 for OMAP4 Changes on top of AOSP source to compile Android 4.4.4 for OMAP4 devices.
Security Updates for Android 4.4.4 R2.0.1 Security-updates which can be added on top of the Android 4.4.4 R2.0.1 source from the Android Open Source Project.
Android 5.1.1 R38 for OMAP4 Changes on top of AOSP source to compile Android 5.1.1 for OMAP4 devices.
Android 6.0.1 R61 for OMAP4 Changes on top of AOSP source to compile Android 6.0.1 for OMAP4 devices
